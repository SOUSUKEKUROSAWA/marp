# Slides by Marp

## Getting Started

```sh
touch ./slides/<file name>.md
```

## How to Export

- `Ctrl + ,`
- `markdown.marp.export`
- `.pdf`または`.pptx`で出力

## How to Add Theme

- select new theme
  - <https://github.com/rnd195/marp-community-themes>
  - <https://github.com/cunhapaulo/marpstyle>
- `Ctrl + ,`
- `markdown.marp.themes`
- add `./themes/*.css`
