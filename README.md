# How to Use
- `cd slides`
- `touch <file name>.md`
```yml
---
marp: true
theme: <theme name>
class: <class name>
---
<!--
headingDivider: 1
-->
<content>
```
- theme name
  - `./themes`内から選択
- headingDivider
  - 1
    - `#`をスライドの区切りと見なす
  - 2
    - `##`をスライドの区切りと見なす
# How to Export
- `Ctrl + ,`
- `markdown.marp.export`
- `.pdf`または`.pptx`で出力
